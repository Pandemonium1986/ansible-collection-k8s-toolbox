Install and configure minikube.

## Requirements

This role is self contained and install minikube via package manager for debian, ubuntu, linux mint, centos.

## Role Variables

None.

## Dependencies

None.

## Example Playbook

```yaml
- name: minikube installation
  hosts: all
  become: true
  tasks:
    - import_role:
        name: pandemonium1986.minikube
```
